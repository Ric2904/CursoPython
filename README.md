# CursoPython
Repositorio con ejercicios y proyectos de Python realizados durante el curso.

## Descripción
Este repositorio contiene ejercicios y proyectos de Python organizados en carpetas independientes. 
Cada carpeta corresponde a un proyecto diferente.

## Proyectos

### 1. Ejercicio1_HolaMundo
- Descripción: Programa que imprime "Hola Mundo".
- Archivo principal: `Ejercicio1_HolaMundo/hola_mundo.py`

### 2. Ejercicio2_Calculadora
- Descripción: Calculadora básica que realiza suma, resta, multiplicación y división.
- Archivo principal: `Ejercicio2_Calculadora/calculadora.py`

### 3. Ejercicio3_Fibonacci
- Descripción: Genera los primeros N números de la serie Fibonacci.
- Archivo principal: `Ejercicio3_Fibonacci/fibonacci.py`
